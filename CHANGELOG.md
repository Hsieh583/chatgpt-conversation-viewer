## [1.0.0] - 2025-12-20

### 新增
- 初始版本發布
- 使用 ijson 串流解析大型 JSON 檔案（支援 250MB+）
- SQLite 資料庫儲存，支援全文搜尋
- Flask Web 介面：
  - 對話列表頁面，支援分頁（每頁 20 筆）
  - 對話詳細頁面，顯示完整對話內容
  - 搜尋功能（搜尋標題和內容）
  - 統計資訊頁面
- Markdown 渲染支援，包含程式碼高亮
- 智慧標籤系統（Coding、Data、Web Development、AI/ML）
- Bootstrap 5 響應式設計
- 記憶體優化設計，適合低記憶體環境（約 2GB 可用記憶體）

### 技術特性
- Python 3.7+ 支援
- 批次寫入資料庫（每 1000 筆提交一次）
- 資料庫索引優化
- 全文搜尋索引（FTS5）
- 使用者和助手訊息視覺區分
- 錯誤處理（404、500 頁面）

### 文件
- 完整的 README.md 使用說明
- 專案結構說明
- 安裝和使用指南
- 故障排除指南
